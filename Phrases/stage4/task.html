Description 

To have more than some predefined spaces, we need to keep them somewhere. In this stage, we will try to store our phrases in database using Room.

Objectives

  1. Create an Entity named Phrase for your phrases with id and phrase as a properties.
  2. Create a class named AppDatabase based on RoomDatabase.  AppDatabse should have a getInstance() and getPhraseDao() methods.
  3. Create a DAO object named PhraseDao, and define methods to insert, delete, get, and getAll phrases from your database.
  4. Update your previous code so that your RecyclerView shows the phrases from the database.
  5. Implement a DialogFragment that shows when you click the addButton and allows you to insert more phrases in the database.
  6. Don't forget to update your RecyclerView: new phrases should pop up immediately!
  7. Update your previous code so that "Delete" TextView deletes the phrase from the screen and database.
  8. Update your previous code so that your notifications take phrases from the database and not from that old list you've used previously. 
  9. Make sure the time formatting for the reminderTv is done properly (Example: 01:01, 11:00, etc.) Use 24-hour format.
  10. Make sure you cannot set reminders with empty database. If user tries to set a reminder with no phrases available, show Toast with an error message and set reminderTv.text to "No reminder set".
